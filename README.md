## Module 28: API, JSON, Data load, dynamic website
## Whats is API ??????

- API হলো এমন একটি Interface যার মাধ্যমে একটি সফটওয়্যার অন্য একটি সফটওয়্যারের সাথে যোগাযোগ করতে পারে।

## JSON কী?

JSON (JavaScript Object Notation) হলো data রাখার এবং আদান-প্রদানের একটা ফরম্যাট।

সহজভাবে 👉
JSON = Data format (text আকারে data সাজানোর নিয়ম)

🧠 কেন JSON দরকার?

যখন Frontend (JavaScript) আর Backend (Server) data আদান-প্রদান করে, তখন তারা JSON ব্যবহার করে।

👉 API থেকে যে data আসে, সেটা বেশিরভাগ সময় JSON format এ থাকে।

📦 JSON দেখতে কেমন?

Example:

{
  "name": "Hridoy",
  "age": 22,
  "isStudent": true,
  "skills": ["HTML", "CSS", "JavaScript"]
}
